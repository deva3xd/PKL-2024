# Requirements
- Php 8
- Composer
- MySQL

# Installation
- clone this repository
- go to directory
- run composer update
- run npm install
- rename .env.example to .env
- run php artisan key:generate
- create database manually using phpmyadmin and adjust the database name like env file
- run php artisan migrate

## Run Project

```bash
  npm run dev
  php artisan serve
```
