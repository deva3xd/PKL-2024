# Sipanduresik
Website Sistem Informasi Pelayanan Administrasi Kependudukan Registrasi Ktp-El

# Persyaratan
- Php 8
- Composer
- MySQL

# Installasi
- clone
- masuk directory
- composer update
- npm i
- rename .env.example ke .env
- isi file .env
- php artisan key:generate
- php artisan migrate

## Cara Menjalankan

```bash
  npm run dev
  php artisan serve
```
